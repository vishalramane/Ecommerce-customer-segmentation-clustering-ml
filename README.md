# 🛒 E-commerce Customer Segmentation & Prediction

## 📌 Overview
This project analyzes e-commerce transaction data to understand customer behavior, segment customers, and predict their future value using machine learning.
It demonstrates a complete end-to-end data analytics workflow.

---

## 🎯 Business Objective
- Segment customers based on purchasing behavior
- Identify high-value and at-risk customers
- Enable targeted marketing
- Visualize insights for stakeholders
- Predict customer value using Machine Learning

---

## 📊 Dataset
- 541,909 transactions
- ~4,300 customers
- Time period: 2010–2011
---

## 🧹 Data Cleaning
- Removed missing Customer IDs
- Removed negative quantities (returns)
- Removed zero-price transactions
- Removed duplicates
- Cleaned text fields
- Created features: Total Amount, Year, Month
---

## 📈 Exploratory Data Analysis
- Revenue trends over time
- Country-wise revenue distribution
- Top-selling products
- Customer purchase behavior
---

## 🧮 RFM Analysis
Customers segmented using:
- Recency
- Frequency
- Monetary value

Segments:
- High Value
- Loyal
- Potential
- At Risk
- Low Value
---

## 🧬 Clustering
Algorithms tested:
- K-Means
- Hierarchical
- DBSCAN
  
Final selection:
👉 K-Means (K=5)

---

## 🤖 Machine Learning
Models used:
- Logistic Regression
- Decision Tree
- Random Forest
- SVM
  
Best Model:
👉 Random Forest 
- High accuracy
- Balanced precision and recall
- Strong interpretability
